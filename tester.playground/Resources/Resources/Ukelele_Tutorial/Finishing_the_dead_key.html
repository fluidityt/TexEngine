Finishing a dead key

When all the output has been entered for your dead key, you need to finish it. If you have the Info inspector visible, you will see a floating window, with the bottom section showing the state stack, looking like this:


Note that there are two states listed, State 0 and none. State 0 is the new dead key state that we just created, and state none is the base state, when no dead keys have been activated. Now, from the Keyboard menu, select “Leave Dead Key State”.


Now observe that the current state stack in the Info inspector shows just “none”, and that the main window shows the normal keyboard, with the new dead key blank, with a black outline to show that it is a dead key.
